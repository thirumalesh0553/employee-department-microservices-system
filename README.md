# Employee-Department Microservices System

## 📌 Overview
This project is a microservices-based application built using Spring Boot and Spring Cloud. It demonstrates service-to-service communication, centralized configuration, and fault tolerance.

## 🏗️ Architecture
- Service Registry (Eureka)
- Config Server
- API Gateway
- Employee Service
- Department Service

## 🔧 Technologies Used
- Java
- Spring Boot
- Spring Cloud
- Eureka Server
- OpenFeign
- API Gateway
- Circuit Breaker (Resilience4j)
- Spring Boot Actuator
- Postman (for API testing)
- Maven
- Git & GitHub

## 🔄 Features
- Microservices architecture
- Service discovery using Eureka
- Centralized configuration using Config Server
- Inter-service communication using OpenFeign
- API Gateway for routing
- Fault tolerance using Circuit Breaker
- Monitoring using Actuator

## ▶️ How to Run
1. Start Service Registry
2. Start Config Server
3. Start API Gateway
4. Start Employee & Department Services
5. Test APIs using Postman

- Add Spring Security (JWT)
